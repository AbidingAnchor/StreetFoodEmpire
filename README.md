# Street Food Empire

An idle mobile game where you build a street food empire — start with a hot dog cart, earn coins over time, unlock stalls, and upgrade to level 10. Built with React, Vite, and Tailwind CSS.

## Development

```bash
npm install
npm run dev
```

```bash
npm run build
npm run preview
```

Deploy on Netlify using the included `netlify.toml` (build: `npm run build`, publish: `dist`).
